## 📖 Overview

This project is a personal portfolio website developed using HTML, CSS, and JavaScript. It was created to present my profile, technical skills, engineering projects, and contact information through a clean and responsive interface. The website is designed to be easy to navigate and simple to maintain, making it suitable for showcasing my work and professional background.

---

## ✨ Features

- Responsive design for desktop and mobile devices
- Modern and clean user interface
- Home, About, Projects, and Contact pages
- Dedicated projects section with GitHub repository links
- Easy to customize and expand
- Simple and lightweight design using HTML, CSS, and JavaScript
